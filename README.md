# E-Shop Brasil - Sistema de Clientes com Big Data

Este projeto simula a gestão de dados de clientes de uma empresa de e-commerce utilizando tecnologias de banco de dados NoSQL e conceitos de Big Data.

- Python
- MongoDB
- Streamlit

- Inserção de clientes
- Listagem de clientes
- Edição de dados
- Exclusão de dados
- Concatenação de nome completo

1. Subir o MongoDB:
- MongoDB (local)

2. Rodar a aplicação:
python -m streamlit run app.py

3. Acessar no navegador:
http://localhost:8501

O sistema permite visualizar, editar e excluir clientes em tempo real.
